## Welcome to Roxas Water Billing System

The **Roxas Water Billing System** is a comprehensive SaaS platform designed to streamline and automate the billing processes of water utility providers. Built to enhance access to water services, this system offers a robust platform for managing billing, payment services, and customer data - ensuring efficient operations for community services, establishments, and organizations.

## What is Roxas Water Billing System?

Roxas Water Billing System is a full-featured water utility management platform that provides:

- **Role-Based Access Control** - Three distinct portals (Admin, Cashier, Meter Reader) with specific permissions and workflows
- **Automated Billing** - Streamlined billing calculations based on water consumption with automated rate application
- **Client Management** - Comprehensive client onboarding with verification and approval workflows
- **Payment Processing** - Integrated payment tracking with cashier confirmation system
- **Meter Reading** - Digital meter reading input with verification workflows
- **Reporting & Analytics** - Data analytics for usage patterns and financial reporting

## Key Features

### User-Friendly Interface
Built with modern web technologies including PHP, JavaScript, jQuery, and Tailwind CSS, the system provides an intuitive interface for both administrators and customers.

### Automated Billing Calculations
The system automatically calculates billing amounts based on:
- Water consumption (current reading - previous reading)
- Rate types (Commercial vs Residential)
- Penalty fees for overdue payments
- Billing cycles and due dates

### Secure Authentication System
Role-based login system with three user designations:
```php
<select name="user_role" id="designation-select">
  <option value="1">Admin</option>
  <option value="2">Cashier</option>
  <option value="3">Meter Reader</option>
</select>
```

### Client Application Workflow
Seamless client addition process with:
1. Initial application encoding
2. Admin review and verification
3. Payment collection by cashiers
4. Final approval and account activation
5. Automatic notifications for status updates

### Document Generation
Print invoices, certificates, and reports using the integrated dompdf library:
- Billing statements
- Disconnection notices
- Payment receipts
- Usage reports

### Real-Time Notifications
Notification system that alerts:
- Admins when clients pay for applications
- Cashiers about new applications requiring processing
- Staff about billing verification needs

## Database Architecture

The system uses a well-structured MySQL database (`wbs_db`) with 22 tables including:

- `users` - User accounts and authentication
- `client_data` - Customer information
- `client_application` - New client applications
- `billing_data` - Billing records and consumption
- `billing_cycle` - Billing periods management
- `transactions` - Payment transactions
- `meter_reading_logs` - Meter reader submissions
- `rates` - Rate configurations
- `penalty_fees` - Late payment penalties
- `notifications` - System notifications

<Note>
  The database schema includes comprehensive logging tables for audit trails and data integrity.
</Note>

## Built For

This system is specifically designed for:

- **Water Utility Providers** - Municipal and private water districts
- **Community Services** - Barangay and local water associations
- **Establishments** - Commercial properties with metered water services
- **Organizations** - Non-profit water service providers
